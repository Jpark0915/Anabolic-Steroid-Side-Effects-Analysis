# Anabolic Steroid Side Effects Analysis

## Overview 💊💊

This exploratory data analysis investigates reported anabolic steroid compounds, user motivations, and side effects using Python and Pandas. The goal was to practice data cleaning, exploratory analysis, and visualization while identifying trends within the dataset.

---

## Tools

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

## Dataset

The dataset contains approximately 800 user-reported observations including:

- Steroid compound
- Reported side effects
- Usage motivation
- Experience level

---

## Project Workflow

- Imported and explored the dataset
- Cleaned missing and inconsistent values
- Parsed list-based side effect columns
- Generated summary statistics
- Built visualizations highlighting common compounds and side effects

---

## Key Findings 💊💊

- Certain compounds appeared more frequently than others.
- Cosmetic and performance motivations were the most common.
- Acne, hair loss, and testosterone suppression were among the most frequently reported side effects.
